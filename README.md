# ai-rag-agent

Learning repo to build a production-grade RAG/agent system.

## Dev quickstart
1) Create venv and install tools (`pytest`, `ruff`, `pre-commit`)
2) `pre-commit install`
3) `pytest`

## Conventions
- Python 3.13+
- Ruff for lint + format (`ruff`, `ruff-format`)
- Tests with `pytest`
